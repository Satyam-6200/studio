# Firebase Studio

This is a NextJS starter in Firebase Studio.

To get started, take a look at src/app/page.tsx.

## Enabling Firebase Authentication

To use the email and password login, you must enable it in your Firebase project:

1.  Go to the Firebase Console.
2.  Select your project.
3.  Navigate to the **Authentication** section from the left-hand menu.
4.  Click on the **Sign-in method** tab.
5.  Find **Email/Password** in the provider list and click the pencil icon to edit.
6.  **Enable** the provider and click **Save**.
